# 古典西洋占星  Skill

当前版本：V1.1

许可证：MIT

面向中文咨询、学习和案例复盘的 skill。它按古典西洋占星的判断链工作，用于本命盘、专题分析、推运、合盘、中点、ACG、校时与案例训练。

它不走泛化星座描述，而是从盘面结构、宫主关系、星体状态、相位接纳和时间技法出发，给出有证据层级的咨询式结论。

## 适用场景

- 本命解读：命格主轴、优势短板、事业方向、财富模式、关系结构。
- 专题判断：事业、财运、婚恋、健康、子女、六亲、自媒体、迁移。
- 推运分析：法达、宫限/小限、黄道释放、返照、太阳弧、次限、三限、行运。
- 关系分析：比较盘、合盘、组合盘、关系阶段、分合窗口。
- 辅助技法：中点、45 度盘、90 度盘、恒星、ACG、迁移盘、Local Space。
- 学习训练：宫主飞星、尊贵无力、接纳互容、中世纪技法、案例复盘与校时。

## 使用方式

1. 下载或克隆本仓库。
2. 在你使用的 AI 工具、Agent 或知识库系统中导入整个文件夹。
3. 让工具读取 `SKILL.md` 和 `references/` 中的规则文档，再按下方示例提出问题。
4. 如果你的工具支持自定义 skill、插件或知识库，可以把本仓库作为一个通用占星分析模块使用。

## 调用示例

```text
用 classical-western-astrology-public 解读这张本命盘。
请按古典西洋占星看事业和财运。
用法达、太阳弧、次限和行运判断未来一年。
请用宫主飞星、接纳和互容分析婚恋结构。
根据已发生事件帮我做出生时间校正。
```

建议先使用 [Almuten](https://www.almuten.net/) 或其他排盘工具生成星盘，再把星盘截图、行星表或宫位表提供给本 skill 使用；这样比只提供出生信息更便于稳定判断。

如果已经有星盘截图，可以直接上传截图并说明问题。资料不完整时，skill 会先标明判断边界，再补问必要信息。

## 判断框架

1. 先读整体盘面：上升、命主星、MC、太阳、月亮、昼夜、角宫星、吉凶星。
2. 再锁定主题宫位：本宫、宫主、宫内星、自然征象星、派生宫。
3. 判断星体状态：庙旺陷弱、尊贵无力、角续果、燃烧、逆行、速度、昼夜得势。
4. 分析关系结构：入相/出相、接纳、互容、定位星、夹制、映点、南北交。
5. 合成证据层级：强证据、混合证据和辅助证据分开说明。
6. 看时间时，先确认本命承诺，再叠加推运系统定位窗口。
7. 输出咨询结论，把术语转译成可理解、可验证、可行动的建议。

## 使用边界

古典占星是一套符号系统和咨询方法，不是科学证伪意义上的预测科学。本 skill 输出应作为象征性分析和反思工具使用，不替代心理咨询、医疗、法律、财务专业判断。

占星判断适合作为象征性分析、趋势整理和咨询参考。涉及健康、法律、投资、生育、未成年人、家庭冲突或重大人生选择时，应结合现实资料与专业意见，避免把单一星象当作决定性结论。

本 skill 会优先给出证据链和不确定性说明，不做死亡断言、疾病诊断、犯罪结论、收益承诺或他人真实意图判定。
