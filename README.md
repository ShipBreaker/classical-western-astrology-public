# 古典西洋占星 Skill 公开分享版

这是一个用于 Codex 的古典西洋占星 skill。它面向中文咨询和学习场景，重点不是太阳星座式描述，而是按古典西洋占星的判断链进行本命、专题、推运、合盘、中点、ACG 和校时分析。

## 适合做什么

- 本命盘整体解读
- 事业、财运、婚恋、健康、子女、六亲、自媒体、学习、迁移等专题判断
- 法达、宫限/小限、太阳弧、次限、三限、返照、行运等推运分析
- 比较盘、合盘、组合盘、关系阶段与分合判断
- 中点、45 度盘、90 度盘、ACG、迁移盘等辅助技法
- 宫主飞星、尊贵无力、接纳互容、恒星、中世纪技法、格局品秩等技术学习
- 案例复盘、校时思路和判断链训练

## 不适合做什么

- 不适合把占星当成医疗、法律、投资或人生决定的唯一依据。
- 不用于断言死亡、疾病诊断、犯罪结论、投资收益或他人真实意图。
- 不适合只凭一个行运、一个相位、一个中点或一条 ACG 线下重大结论。
- 不提供娱乐化、泛化的星座运势文。

## 安装方式

1. 解压分享包，保留文件夹名 `classical-western-astrology-public`。
2. 将整个文件夹放入 Codex 的 skills 目录。
3. 重新打开 Codex，或开启一个新会话，让 skill 被重新发现。
4. 在对话中使用相关触发词，例如：
   - `用 classical-western-astrology-public 解读这张本命盘`
   - `请按古典西洋占星看事业和财运`
   - `用法达、太阳弧、次限和行运判断未来一年`
   - `请用宫主飞星和接纳分析婚恋结构`

## 使用时需要提供的信息

做完整本命或推运时，最好提供：

- 出生日期
- 准确出生时间
- 出生地
- 出生时间可靠度
- 星盘图、行星表或宫位表
- 使用的黄道和宫制，如果已知
- 具体问题，例如事业、婚恋、财运、健康、迁移、校时等

如果已经有星盘截图，可以直接上传截图并说明想问的问题。若资料不足，skill 会先说明限制，再补问必要信息。

## 核心判断方法

这个 skill 的默认顺序是：

1. 先看整体结构：上升、命主星、MC、太阳、月亮、昼夜、角宫星、吉凶星。
2. 再定主题宫：本宫、宫主、宫内星、自然征象星、派生宫。
3. 判断星体状态：庙旺陷弱、尊贵无力、角续果、燃烧、逆行、速度、昼夜得势。
4. 判断相位关系：入相/出相、接纳、互容、定位星、夹制、映点、南北交。
5. 合成重复证据：强证据、混合证据和辅助证据分开说。
6. 若看时间，先确认本命承诺，再叠加推运系统。
7. 最后给出咨询式结论，而不是只罗列术语。

## 常用输出格式

默认回答通常会采用：

```markdown
**盘面总纲**
一句话说明命格主轴、强弱、最关键的宫位/星体/结构。

**判断链**
- 证据 1：宫位 / 宫主 / 星体状态 -> 含义
- 证据 2：相位 / 接纳 / 飞星 / 推运 -> 含义
- 证据 3：重复证据或案例逻辑 -> 含义

**结论**
给出可理解、可执行的解读；强证据先说，弱证据标明为辅助。

**边界**
说明出生时间、宫制、资料缺口、现实条件或高风险主题的限制。
```

## 使用建议

- 问题越具体，判断越清楚。比如“我适合什么职业路径”比“帮我看命”更容易得到有效答案。
- 推运问题最好给出目标年份、已发生事件和想验证的主题。
- 合盘问题最好分别提供双方本命信息，并说明关系阶段。
- 校时问题需要多个已发生事件，例如搬家、升学、入职、离职、结婚、分手、手术、亲属事件、公开成就等。
- 如果涉及健康、法律、投资、生育或重大人生选择，应把占星判断当作象征性参考，并结合现实专业意见。
